## 安装

```
pnpm add roodash
```

## 用法

这是一个非常简单的用法例子，更多用法请查看完整文档 [完整文档](https://jxxuxuefeng.github.io/roodash/).

```ts
import { isEmpty, getFileExtension } from 'roodash'

isEmpty({}) 
// => true

getFileExtension('file.txt')
// => 'txt'

```
