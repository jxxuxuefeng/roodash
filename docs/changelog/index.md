# 更新日志
`roodash` 遵循 [Semantic Versioning 2.0.0](http://semver.org/lang/zh-CN/) 语义化版本规范。

#### 发布规范
- 修订版本号：bugfix 更新
- 次版本号：带有新特性的向下兼容的版本
- 主版本号：含有破坏性更新和新特性
